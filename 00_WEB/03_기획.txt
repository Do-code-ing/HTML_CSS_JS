3. 기획: 어떻게 동작해야 하는가?
    UI 모델링
        UI: User Interface
        - 사용자가 시스템을 제어 및 사용하는데 사용되는 조작 장치
        - 검색 버튼과 같은 어떤 버튼들, 기능들

        모델링:
        - 직접 만들기 전에 가상으로 만들어 보는 것
    
    UI 모델링은 '사용자 입장에서 동작하는 기능을 가상으로 만들어 보기'라고 할 수 있다.
    중요한 이유: 커뮤니케이션 - 무엇을 만들건지 서로가 공유하기 위해서
    (어떤 기능을 넣고, 그 기능을 어디에 배치하고 등등)

    모델링하기
        1. 손으로 그리기
        2. Pencil (http://pencil.evolus.vn/, http://opentutorials.org/course/221/1416)
        3. balsamiq (http://balsamiq.com) (유료)
        4. PowerPoint
